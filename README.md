To-Do List Dashboard

A modern and responsive To-Do List Dashboard built using HTML, CSS, and JavaScript. This application helps users manage daily tasks efficiently by providing features such as task creation, editing, deletion, status tracking, priority management, deadline reminders, search functionality, and progress statistics.

Features
Add new tasks with deadlines
Edit existing tasks
Delete tasks
Mark tasks as completed
Track task status (Pending, In Progress, Completed)
Assign task priorities (Minor, Normal, Critical)
Set deadline date and time
Highlight overdue tasks
Search tasks instantly
View completion statistics and progress
Persistent data storage using Local Storage
Fully responsive dashboard design

--Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsive design
JavaScript (ES6) – Application logic and interactivity
Font Awesome – Icons used throughout the interface
Local Storage API – Client-side data persistence

--How to Run the Project
Clone the repository:
git clone https://github.com/your-username/to-do-dashboard.git
Navigate to the project folder:
cd to-do-dashboard
Open the index.html file in your browser.

No additional installations or dependencies are required.

--Usage
Click "Add New Task" to create a task.
Enter the task title, status, priority, and deadline.
Save the task to display it on the dashboard.
Use the checkbox to mark tasks as completed.
Click the edit icon to update task details.
Click the delete icon to remove tasks.
Use the search bar to find specific tasks quickly.
--Dashboard Statistics

The application automatically displays:

Total number of tasks
Number of completed tasks
Number of pending tasks
Task completion percentage
--Data Storage

All tasks are stored in the browser using Local Storage. This means your tasks remain available even after refreshing or reopening the browser.
